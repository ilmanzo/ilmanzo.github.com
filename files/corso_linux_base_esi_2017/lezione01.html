it's a kind of magic

#nientepanico

$ fdisk -l 2016-02-26-raspbian-jessie.img
Disk 2016-02-26-raspbian-jessie.img: 3,8 GiB, 4029677568 bytes, 7870464 sectors
Units: sectors of 1 * 512 = 512 bytes
Sector size (logical/physical): 512 bytes / 512 bytes
I/O size (minimum/optimal): 512 bytes / 512 bytes
Disklabel type: dos
Disk identifier: 0x0cf63fa8

Device                          Boot  Start     End Sectors  Size Id Type
2016-02-26-raspbian-jessie.img1        8192  131071  122880   60M  c W95 FAT32 (LBA)
2016-02-26-raspbian-jessie.img2      131072 7870463 7739392  3,7G 83 Linux

quindi: la prima partizione inizia al byte n. 8192 * 512 = 4194304

e la seconda partizione inizia al byte n. 131072 * 512 = 67108864

impareremo questi comandi prossimamente

#cd /home/andrea/projects/Raspberry Pi Emulator/SharedSupport

#mkdir  /tmp/raspi/one /tmp/raspi/two

#mount -v -o offset=4194304 -t vfat 2016-02-26-raspbian-jessie.img  /tmp/raspi/one

#mount -v -o offset=67108864 -t ext4 2016-02-26-raspbian-jessie.img /tmp/raspi/two
