l'istruzione while e il concetto di ciclo

ogni ciclo ha bisogno della condizione di uscita, altrimenti si ripetera' per sempre, e il computer sembra "bloccato"

n=0
while True:
  if n>30:
    break
  n=n+1
  print(n)
print("Finito!")

l'istruzione break serve per uscire da un ciclo in qualsiasi momento

9/24

l'istruzione while e il concetto di ciclo

oltre al break abbiamo il continue che serve per interrompere l'iterazione e "ritornare" all'inizio del ciclo

n=0
while True:
  if n>30:
    break
  n=n+1
  if n%2==0:
    continue
  print(n)
print("Finito!")

aiuto:

per sapere se un numero e' divisibile per un altro, usare l'operatore modulo '%' che ritorna il resto della divisione. Quindi

8%2 == 0 perche' 8/2 = 4 con il resto di 0

9%2 == 1 perche' 9/2 = 4 con il resto di 1

funzioni in Python: definizione

in Python, le funzioni si definiscono con l'istruzione def

sono utili per raccogliere dei blocchi di codice che poi possiamo riutilizzare piu' volte

def saluta():
  print("ciao")
  print("sto imparando Python")

NB: def serve solo per stabilire come e' fatta e cosa fa la funzione, ma non la esegue! Le istruzioni che stanno dentro la funzione vengono eseguite effettivamente solo quando la richiamo:

def saluta():
  print("ciao")
  print("sto imparando Python")

variabili nelle funzioni

una funzione puo' accettare uno o piu' parametri in ingresso per poter essere riutilizzata in occasioni diverse

def abs(numero)
    if numero>=0:
        print(numero)
    else:
        print(-numero)

abs(35)
abs(-22)

NB: le variabili tra parentesi nella definizione hanno valore simbolico ed "esistono" solo all'interno della funzione al di fuori di abs(), la variabile numero non esiste

valori di ritorno nelle funzioni

una funzione puo' anche ritornare un valore, che puo' essere utilizzato nel programma che esegue la funzione. Per ritornare un valore, si usa l'istruzione return

def area_rettangolo(base,altezza):
    return base*altezza

def area_quadrato(lato):
    return lato**2

def area_cerchio(raggio):
    return 3.141516*raggio*raggio


raggio = 4
lato = 7
cerchio = area_cerchio(raggio)
quadrato = area_quadrato(lato)
print("area del cerchio di raggio",raggio,"=", cerchio)
print("area del quadrato di lato",lato,"=", quadrato)
