shell script

quando lavoriamo sulla shell scriviamo tutti i comandi in modo interattivo, cioe' aspettando ogni volta l'esecuzione prima di dare il successivo; possiamo anche creare una sequenza di comandi mettendoli tutti di seguito dentro un file di testo, che se rendiamo eseguibile diventa uno shell script

dentro gli shell script possiamo usare variabili, istruzioni di controllo, cicli come se fosse un linguaggio di programmazione

creando degli script sostanzialmente possiamo crearci nuovi comandi

esempio di script

#!/bin/sh
#questo script scansiona un documento in pdf e lo spedisce via mail
#aspetto ok dell'utente prima di proseguire
read -p "premi ENTER per iniziare la scansione"
#ricavo la data e l'ora attuale
TIMESTAMP=$(date +%Y%m%d%H%M%S)
#scanimage fa parte del pacchetto 'sane' https://it.wikipedia.org/wiki/SANE
scanimage --progress --buffer-size=256 --format=tiff \
  --resolution 100 --mode Gray > /tmp/$TIMESTAMP.tiff
#mogrify e convert sono nel pacchetto imagemagick
mogrify -normalize -level 10%,90% -sharpen 0x1 /tmp/$TIMESTAMP.tiff
convert /tmp/$TIMESTAMP.tiff /home/andrea/scansioni/$TIMESTAMP.pdf
#do un messaggio all'utente e spedisco via mail
echo File salvato come /home/andrea/scansioni/$TIMESTAMP.pdf
echo "ciao, in allegato il documento scansionato" | mail -s "scan del $TIMESTAMP" \ 
    ilmanzo@gmail.com -A /home/andrea/scansioni/$TIMESTAMP.pdf

un semplice servizio di rete

xinetd e' un servizio che fornisce internamente semplici demoni di rete.

apt-get install xinetd
vi /etc/xinetd.d/daytime  # metto disable a 'no'
systemctl restart xinetd    # avvio il servizio
cat /etc/services|grep daytime
apt-get install telnet 
telnet 127.0.0.1 daytime

xinetd permette anche l'utilizzo via rete dei programmi concepiti per un I/O locale.

19/23

un semplice servizio di rete

con xinetd possiamo rendere "network aware" qualsiasi script che legga da stdin e scriva su stdout

nano /etc/xinetd.d/testservice

service testservice
{
  port            = 54321
  socket_type     = stream
  protocol        = tcp
  wait            = no
  user            = root
  server          = /usr/local/bin/mioscript.sh
  server_args     = test
}

cat /usr/local/bin/mioscript.sh

#!/bin/sh
echo "<html>ciaooo</html>"

aggiungo testservice in /etc/services

testservice     54321/tcp
