funzioni in Python: definizione

in Python, le funzioni si definiscono con l'istruzione def

digressione: funzione fattoriale

un po' di matematica...

n! = n(n-1)(n-2)(n-3) ... 2*1

quindi 4! = 4*3*2*1 = 24

quindi 5! = 5*(4*3*2*1) = 120

ovvero 5! = 5*4!

n! = n*(n-1)!

... ma 0! = 1

def factorial(n):
  if n == 0:
      return 1
  else:
      return n*factorial(n-1)

quando una funzione richiama se stessa, viene detta ricorsiva

28/49

Cancellazione di liste

Usare le porzioni per cancellare elementi delle liste non poi cos pratico ed e' facile sbagliare. Python fornisce un'alternativa molto pi leggibile.

del rimuove un elemento da una lista:

>>> a = ['uno', 'due', 'tre']
>>> del a[1]
>>> a
['uno', 'tre']

Come puoi facilmente immaginare, del gestisce anche gli indici negativi e avvisa con messaggio d'errore se l'indice al di fuori dei limiti ammessi.

Puoi usare una porzione come indice di del:

>>> lista = ['a', 'b', 'c', 'd', 'e', 'f']
>>> del lista[1:5]
>>> print(lista)
['a', 'f']

funzione con parametri lista

dizionari

Un dizionario python e' un 'contenitore di dati'. Simile a un magazzino, o per gli informatici, a un database, permette di accedere a un dato non solo per indice numerico (come nelle liste) ma anche per "chiave" , dove la chiave puo' essere una stringa o qualsiasi altro dato purche' immutabile. Esempio:

giornidelmese={'Gen':31, 'Feb':28, 'Mar':31, 'Apr':30, 'Mag': 31}
g=giornidelmese['Mar']
print(g)

per semplificare, possiamo iniziare a pensarlo come una lista che usa come indici le stringhe anziche' i numeri.
