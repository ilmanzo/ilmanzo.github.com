LAMP stack

cos' LAMP ?

Si parla di "STACK" in quanto LAMP un software che fornisce una infrastruttura completa per una applicazione web based, dalla richiesta HTTP (layer di rete) fino alla logica di elaborazione e visualizzazione dati (layer di presentazione).

https://en.wikipedia.org/wiki/LAMP_(software_bundle)

installazione LAMP

usando il package manager 'apt' su Linux:

sudo apt update
#installa apache (webserver) e mysql server (database)
sudo apt install apache2 mysql-server
#per installare php:
sudo apt install php libapache2-mod-php php-mcrypt php-mysql
#opzionali ma consigliati:
sudo apt install mysql-client php-cli

per pc Windows, esistono dei software che 'impacchettano' tutto il necessario in un unico installante, ad esempio WAMP oppure XAMPP. Questi software contengono anche un programma di configurazione e tutte le librerie aggiuntive necessarie

cos'e' NODE ?

node.js e' una piattaforma opensource "event-driven" per sviluppare applicazioni basate sul linguaggio JavaScript.

Node.js usa un modello I/O non bloccante e ad eventi, che lo rende un framework leggero ed efficiente. L'ecosistema dei pacchetti di Node.js si chiama npm, il pi grande insieme di librerie open source al mondo.

https://nodejs.org/it/

esempio: un server web creato con node.js

//questo e' codice javascript
var http = require('http');

var server = http.createServer(function (req, res) {
  res.writeHead(200, {'Content-Type': 'text/plain'});
  res.end('Hello World');
})

server.listen(1337, '127.0.0.1');

console.log('Server running at http://127.0.0.1:1337/');

//salvare ed eseguire da shell con
$ node helloworld.js
//collegarsi con il browser all'indirizzo http://127.0.0.1:1337

cos'e' Node-RED ?

Node-RED e' un ambiente integrato; sviluppato inizialmente da IBM (2013), e poi rilasciato come opensource, per creare progetti che si basano sul flow programming (programmazione espressa come flussi di dati, termine coniato da J. Paul Morrison, periodo 1970).

Node-RED adotta il concetto di "flow" per permettere all'utente di disegnare graficamente la logica dell'elaborazione e stabilire gli input e gli output, in un modo molto familiare a chi gi lavora con l'elettronica, circuiti e segnali.

Node-RED si appoggia su nodejs per funzionare ed distribuito come modulo NPM.

tra i punti di forza di Node-RED, la sua semplicita' e il fatto che una vasta comunita' di sviluppatori metta a disposizione liberamente moduli aggiuntivi e flussi per le piu' diverse esigenze

Social Development: i flussi possono anche essere importati ed esportati, come se fossero veri e propri sorgenti

la documentazione e' chiara ed esaustiva, e fa capo al sito https://nodered.org/

bonus: molto istruttivo il video dello sviluppatore originale Nick OLeary https://nodered.org/about/

come si installa ?

ci sono varie opzioni per usare node-RED:

  • installazione locale: essendo Node-RED distribuito come package di nodejs, va installato con il package manager di nodejs, ovvero npm

  • installazione condivisa: si pu installare su un server in rete e tutti gli utenti useranno un browser per accedere a quella installazione

  • usare una immagine Docker

  • installazione su un device dedicato, come Raspberry PI o un qualsiasi dispositivo Android
  • usare una installazione cloud messa a disposizione da terzi (AWS, Azure, IBM Bluemix ...)
