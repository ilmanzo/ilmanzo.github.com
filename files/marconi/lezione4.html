node-red all'avvio

per far partire automaticamente Node-RED all'avvio del sistema, dovremo usare un systemd service, o nei sistemi più vecchi, uno script di init. 

[Unit]
Description=Node-RED
After=syslog.target network.target

[Service]
ExecStart=/usr/bin/node-red --max-old-space-size=128 -v
Restart=on-failure
KillSignal=SIGINT

# log output to syslog as 'node-red'
SyslogIdentifier=node-red
StandardOutput=syslog

# non-root user to run as
WorkingDirectory=/home/andrea/
User=andrea
Group=andrea

[Install]
WantedBy=multi-user.target

il nodo "function"

un nodo di tipo "funzione" e' semplicemente una funzione javascript che riceve un oggetto 'message' e ritorna un oggetto 'message' , modificandolo o creandolo ex novo. Esempio:

// Crea un oggetto Data a partire dal payload
var date = new Date(msg.payload);
// modifica il payload
msg.payload = date.toString();
// ritorna a sua volta il messaggio al nodo successivo
return msg;

https://nodered.org/docs/writing-functions
