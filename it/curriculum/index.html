Esperienza Lavorativa#

  • 2022-oggi: QE Automation Engineer, SUSE
    Sviluppo di test automatizzati e garanzia di qualità (QA) per il rilascio dei prodotti SUSE. Manutentore di alcuni pacchetti RPM. Contributore open source.

  • 2017-2022: cloud & automation senior sw engineer, Value Transformation Services
    Lavoro su un’implementazione cloud ibrido completamente automatizzata. Consegna e gestione, tramite best practice DevOps, di una piattaforma personalizzata utilizzata dal cliente per richiedere soluzioni IaaS e PaaS.

  • 2013-2016: linux senior system engineer, Value Transformation Services
    Gestione e introduzione dell’automazione completa su un parco di circa 18.000 server Linux multi-datacenter.

  • 2011-2013: linux system engineer, Unicredit

  • 2005-2011: sviluppatore di prodotti di remote banking e pagamenti remoti per il gruppo Unicredit
    Lavoro su prodotti di remote banking, sia su tecnologie desktop che POS.

  • 2001-2005: amministratore di rete e sicurezza in Quercia Software spa
    Amministrazione di dispositivi LAN e WAN, sicurezza e implementazione di infrastruttura VoIP.

  • 2000-2001: sysadmin, network administrator presso un Internet Service Provider locale, Netbusiness spa
    Gestione di servizi ISP, rete e operazioni.

  • 1999-2000: sviluppatore, DBA Oracle e sysadmin VAX/VMS
    Implementazione di personalizzazioni e migrazioni sul software di gestione bibliotecaria più grande del mondo.

  • Attivo nel settore IT dal 1993; primo server Linux installato nel 1995.
