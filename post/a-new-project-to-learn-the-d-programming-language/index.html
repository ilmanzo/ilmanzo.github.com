A new project to learn the D Programming Language

learn D programming language by solving simple exercises
December 7, 2014 · Andrea Manzini

I’ve started a new side project named D Lang Koans, it’s a simple series of exercises organized in a unit test suite.

The “koans” are heavily inspired by similar projects for other languages, but I didn’t found anything similar for D.

I’ll try to maintain and evolve it in the spare time, I hope you’ll find it useful.
